# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest release | Yes |
| Older releases | No |

Only the latest release is supported. Once a stable release has been published, only the latest stable release will receive security updates.

## Reporting a Vulnerability

Do not report security vulnerabilities through public GitHub issues.

Report a vulnerability through GitHub's security advisory feature:

https://github.com/gabrielhamalwa/storybook-symfony-bundle/security/advisories/new

Include a description of the issue, the steps taken to create it, the affected versions, and, if known, mitigations. Reports will be reviewed as soon as possible.

## Preferred languages

English is the preferred language for all communications.
